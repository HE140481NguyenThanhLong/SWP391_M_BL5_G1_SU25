Cách sử dụng Breadcrumb Component

1. Copy HTML: Sao chép đoạn HTML của breadcrumb phù hợp với trang của bạn

2. Cập nhật links: Thay đổi href="#" thành đường dẫn thực tế

3. Cập nhật text: Thay đổi text trong các thẻ a và span

4. Current page: Trang hiện tại sử dụng thẻ span với aria-current="page"

5. Responsive: Component tự động responsive trên mobile và desktop
